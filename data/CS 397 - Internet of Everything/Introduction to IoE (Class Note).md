# Semiconductor and Its Application

> Valence band is where free electrons of a atom resides

The P-N junction, P stands for proton/positive, N stands for negative/neutron.

The P contains electrons, and N contains protons & holes.

In a diode, where the P and N meets, there is a depletion layer where electrons and the holes combine which separate the two nodes.

One of the main application for diode is rectifying (AC to DC conversion). When AC flows through a diode, when it is positive, the depletion layer narrows and allows the electricity to flow through. When it is negative, the depletion layer widens (P is negative and N is already negative, thus they repel each other away increasing the space between the two) and electricity cannot go through.
